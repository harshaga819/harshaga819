# Featured Projects

### 1. PDF Chatbot

An AI-powered document question-answering application built using LangChain, FAISS, Hugging Face Embeddings, Groq LLM, and Streamlit. Users can upload PDF files and ask questions using natural language through a Retrieval-Augmented Generation (RAG) pipeline.

---

### 2. ChatWithWebsite

An AI application that enables users to chat with any public website by simply providing its URL. The project extracts website content, builds embeddings, stores them in FAISS, and answers questions using LangChain and Groq LLM.

---

### 3. JobScan – Fake Job Detection

A Machine Learning and NLP project that detects fraudulent job postings using TF-IDF feature extraction and multiple classification algorithms, helping users identify potentially fake job listings.

---

### 4. Facial Emotion Recognition System

A Deep Learning and Computer Vision project that recognizes human emotions from facial expressions in real time using TensorFlow, OpenCV, and MobileNetV2 Transfer Learning.

---

### 5. RentWise – Rent Price Prediction

A machine learning web application that predicts house rental prices based on location, BHK, area, floor, and other property attributes using XGBoost.

---

### 6. IPL 2025 Auction Analysis

Performed web scraping and data analysis to explore player performance, auction trends, and team strategies using Python and data visualization.

---

### 7. Road Accident Data Analysis

Analyzed road accident datasets to identify accident patterns, contributing factors, and safety insights through exploratory data analysis and visualization.
